# Mythology Primer

Quick reference for mythological traditions used in Kojima Building Principles.

---

## Iroquois (Haudenosaunee) Creation

### The Story

Sky Woman falls from celestial realm through a hole near the Great Tree. Birds catch her; she lands on Great Turtle's back floating in primordial waters. Animals dive to bring up mud, creating Turtle Island (North America). 

Sky Woman's daughter grows, becomes pregnant by the West Wind, and dies giving birth to twin sons:
- **Sapling (Tharonhiawakon)** - Creates good things: sun, moon, stars, plants, animals, humans
- **Flint (Tawiscaron)** - Creates challenging things: predators, thorns, disease, harsh terrain

The twins are not good vs. evil—both are *necessary*. Their conflict shapes the world.

### Key Elements for Game Design

| Element | Meaning | Game Application |
|---------|---------|------------------|
| **Sky Woman** | Bridge between celestial and earthly | Protagonist spanning two realms/states |
| **The Fall** | Transition, entering new world | Inciting incident, entering game-world |
| **Great Turtle** | Foundation, steady support | The world itself; stable base systems |
| **The Dive** | Sacrifice to create, retrieving from depths | Core gathering/delivery mechanics |
| **Turtle Island** | World built from nothing through cooperation | World-building, reconstruction themes |
| **Sapling** | Creator, connector, life-bringer | Builder characters, constructive mechanics |
| **Flint** | Necessary opposition, challenge-creator | Antagonists (not evil), obstacle systems |
| **Twin Snakes** | Duality from single source | Character pairs, split paths, genetic themes |
| **Great Tree of Peace** | Unity, connection, shared foundation | Network systems, faction unity |
| **The Mother's Death** | Sacrifice enabling new creation | Mentors who die, costs of progress |

### Cross-Reference Rules

- Sapling and Flint MUST share origin (same source, divergent paths)
- Neither twin purely good or evil
- Both twins necessary for complete world
- Cooperation (the diving animals) is how world is built

## Egyptian Cosmology

### Core Concepts

| Concept | Egyptian Meaning | Pronunciation |
|---------|------------------|---------------|
| **Ka** | Life force, vital essence (stays with body) | "kah" |
| **Ba** | Personality, soul (travels after death) | "bah" |
| **Akh** | Transformed spirit (Ka + Ba reunited) | "ahkh" |
| **Duat** | Underworld, realm of dead | "doo-aht" |
| **Ma'at** | Cosmic balance, truth, justice, order | "mah-aht" |
| **Isfet** | Chaos, injustice, disorder (opposite of Ma'at) | "is-fet" |

### Key Figures

| Figure | Role | Game Application |
|--------|------|------------------|
| **Osiris** | Dies, is resurrected, rules underworld | Protagonist death/rebirth, transformation |
| **Isis** | Resurrects Osiris, powerful magic | Healer/resurrector characters |
| **Set** | Kills Osiris, god of chaos—not purely evil | Antagonist bringing necessary change |
| **Horus** | Son who avenges, falcon-headed | Next generation, inheritance themes |
| **Anubis** | Guides souls through death, jackal-headed | Guide characters, death-navigation |
| **Thoth** | Wisdom, writing, knowledge keeper | Lore characters, information systems |
| **Ammit** | Devours unworthy hearts | Fail-state, consequence of imbalance |

### The Weighing of the Heart

After death, the heart is weighed against Ma'at's feather:
- Heart lighter than feather = passage to afterlife
- Heart heavier = devoured by Ammit

**Game Application:** Judgment systems, moral weight, consequence assessment

### The Duat (Underworld)

- Not punishment—transformation space
- Personal history manifests as landscape
- Guides appear to assist
- Tests must be passed
- Return possible for the worthy

**Game Application:** Liminal spaces, dream sequences, death mechanics, "the Beach"

### Key Elements for Game Design

| Element | Meaning | Game Application |
|---------|---------|------------------|
| **Ka/Ba** | Soul components | Split identity, death mechanics |
| **Duat** | Liminal passage realm | Between-death space, dream worlds |
| **Osiris Cycle** | Death → rebirth → transformed | Respawn as narrative, character growth |
| **Ma'at/Isfet** | Balance vs chaos | Consequence systems, world-state |
| **Weighing** | Actions have weight | Moral systems, judgment |
| **The Two Lands** | Upper/Lower Egypt unified | Divided worlds seeking unity |
| **Scarab** | Rebirth, pushing sun, cycles | Cyclical gameplay, renewal |

### Cross-Reference Rules

- Death must enable transformation, not just reset
- Balance (Ma'at) requires both order AND chaos
- Judgment should feel cosmic, not arbitrary
- The underworld is traversed, not punished

## World Tree Archetype

### Universal Structure

Found across cultures: Yggdrasil (Norse), Great Tree of Peace (Iroquois), Tree of Life (Mesopotamian), Bodhi Tree (Buddhist), etc.

```
    CROWN/BRANCHES
    ─────────────── Heaven / Spirit realm / Divine
          │
        TRUNK
    ─────────────── Middle world / Human realm / Present
          │
     ROOTS
    ─────────────── Underworld / Death realm / Past
```

### Key Properties

| Property | Meaning | Game Application |
|----------|---------|------------------|
| **Connects realms** | Enables passage between worlds | Fast travel, dimensional systems |
| **Has roots everywhere** | Invisible connections underground | Network infrastructure, hidden links |
| **Bears fruit** | Source of knowledge/power | Collectibles, upgrades, revelations |
| **Can be damaged** | World-tree health = world health | World-state systems |
| **Is axis mundi** | Center around which world turns | Central location, hub design |

### Game Applications

- **Network Systems:** The tree as information/connection network (Chiral Network, internet, AI)
- **Hub Design:** Central location connecting all regions
- **Faction Unity:** Groups connected through shared "roots"
- **Progression:** Climbing the tree = advancing through realms

## Synthesis Guidelines

### Combining Traditions

Kojima creates synthetic pantheons. When combining:

1. **Identify complementary elements**
   - Iroquois provides creation structure
   - Egyptian provides death/rebirth mechanics
   - Together: complete life-death-rebirth cycle

2. **Avoid contradiction**
   - Don't combine mutually exclusive cosmologies
   - Or acknowledge contradiction as theme (gnosticism!)

3. **Layer, don't blend**
   - Different traditions can govern different systems
   - Creation myth for world-building
   - Death mythology for respawn/consequence
   - Hero's journey for narrative structure

4. **Maintain invisibility**
   - Never name sources in-game
   - Let scholars discover
   - Players should feel, not read

### Example Synthesis

**Death Stranding Pantheon:**
| Layer | Tradition | Elements |
|-------|-----------|----------|
| World structure | Iroquois | Turtle Island rebuilt, diving for creation |
| Death mechanics | Egyptian | Beach as Duat, Ka/Ba, Osiris return |
| Connectivity | World Tree | Chiral Network as roots and branches |
| Apocalypse frame | Christian | Death Stranding as flood, Bridges as ark |
| Physics | Scientific | Big Bang, extinction events, entanglement |

## Cultural Sensitivity Notes

These are living traditions. When using:

- **Research deeply** - superficial borrowing creates stereotype
- **Respect sacred elements** - some things shouldn't be gamified
- **Consult when possible** - especially for less documented traditions
- **Transform, don't copy** - Kojima inverts Ka/Ba deliberately
- **Never mock** - use traditions as structural wisdom, not exotic flavor

### Traditions Requiring Extra Care

- Indigenous traditions (including Iroquois) - living cultures with sovereignty
- Religions with active practitioners - avoid depicting worship incorrectly
- Marginalized traditions - avoid reinforcing stereotypes
- Funerary practices - handle death customs respectfully
